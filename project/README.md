# Zomato Dataset Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Zomato dataset to understand restaurant trends, customer preferences, ratings, pricing, and online delivery services.

---

## Objective
The main objective of this project is to:
- Analyze restaurant data
- Identify trends and patterns
- Perform data cleaning
- Generate visual insights using charts and graphs

---

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

---

## Dataset Information
The dataset contains information about:
- Restaurant names
- Ratings
- Votes
- Cost for two
- Online delivery
- Table booking
- Restaurant types
- Locations

---

## Data Cleaning Steps
- Checked missing values
- Removed duplicates
- Handled null values
- Corrected data types
- Renamed columns

---

## Exploratory Data Analysis
The following analysis was performed:
- Rating distribution
- Online delivery analysis
- Cost analysis
- Restaurant category analysis
- Customer voting trends

---

## Visualizations
The project includes:
- Bar charts
- Count plots
- Histograms
- Pie charts
- Heatmaps

---

## Key Insights
- Most restaurants have ratings between 3.5 and 4.0
- Online delivery is highly popular
- Mid-range restaurants dominate the dataset
- Higher ratings often correlate with more votes

---

## Conclusion
This project helped in understanding restaurant business trends and customer preferences using Exploratory Data Analysis techniques.
